# Examples

## Coding Request

Input:

```text
帮我做一个能上传 PDF 题库然后刷题的网站。
```

Output focus:

- Intent: build a study web app
- Inputs: text-based PDF files
- Outputs: extracted questions, practice mode, wrong-answer review
- Open questions: OCR support, question formats, local-only or cloud sync
- Safety: use mock/public files only

## Evaluation Request

Input:

```text
Turn this AI use case into eval test cases.
```

Output focus:

- Normal case
- Edge case
- Unsafe input case
- Expected behavior
- Pass/fail rubric

## Analysis Request

Input:

```text
帮我检查这个 portfolio report 有没有逻辑问题。
```

Output focus:

- Treat data as mock/public unless confirmed
- Avoid investment advice
- Check unsupported claims, inconsistent assumptions, missing evidence, and unclear definitions
