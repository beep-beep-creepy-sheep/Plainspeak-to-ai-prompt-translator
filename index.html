Static prompt-engineering demo

PlainSpeak Prompt Translator

Use a local LLM to understand messy human requests, then rewrite them as structured, safer prompts.

Ollama is the real compiler for open-ended requests. Rule-based mode is only a no-setup fallback and will miss many user intents.

Ollama setup checklist
  1. Open the Ollama Mac app, or run ollama serve.
  2. Run ollama list and copy an installed model name.
  3. Paste that exact name into the model box, for example llama3.1:latest.
  4. If the live site fails because of browser security, run this app locally with python3 -m http.server 8017.
